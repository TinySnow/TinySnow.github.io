```diff
@@ -0,0 +1,47 @@
+# 这是我对所有迷茫的人的建议
+
+　　试。
+
+　　就这一个字。
+
+　　我个人把“迷茫”这种状态定义为：不知道前往何方，但是总感觉不能留在原地空耗时间，一种前怕狼后怕虎、犹犹豫豫的状态。
+
+　　虽然“尝试”并不能解决走向何方的问题，但它可以辅助解决排除选项。
+
+　　迷茫这种状态的潜在原因之一，是眼前的可选项太少。好像每个都不是自己想要的、想做的，但自己真正想要和想做的又不知道是什么。
+
+　　那就只能试。
+
+　　福尔摩斯有一句至理名言：“排除了所有不可能的，剩下的即使多么不可思议，那也一定是真相。”
+
+　　而尝试这个事情，无论是名人还是凡人，都可以做。
+
+　　投资界大师索罗斯，狙击英镑把英国央行打垮这位，他一贯的投资风格就是无论好不好，先放一点资金试试水。如果情况不对，马上撤回资金，跟巴菲特风格迥异。而每次他后面发现自己做了一个错误的投资决定时，就会出现严重的背疼。当然这都是奇闻轶事了。
+
+　　索罗斯都在尝试，他也不会了解所有东西，所以我们没必要担心尝试会为我们带来什么不好的后果。
+
+　　但尝试有一个不高不低的门槛：勇气。
+
+　　比如让人去跳伞、蹦极、潜水等等，这些东西可以经过训练而脱敏，但每个人的第一次，终究都是会害怕的。
+
+　　这是尝试的唯一门槛。
+
+　　尝试不能指导你做什么最好，它只是一个很好的辅助。
+
+　　当我们把所有尝试后不感兴趣的事情都删除，理智会告诉我们，剩下的选项中，无论再没有感觉，那也比做那些让自己感到痛苦的事情好得多。
+
+　　做饭、运动、摄影、跳舞、绘画、唱歌、学习、讲课、写作、交际等等等等，而运动中有那么多分项、学习中有那么多学科，谁又能保证自己一定不会喜欢其中的某一种东西呢？
+
+　　这需要花费或短或长、甚至有可能漫长的时间去寻找，但那又如何？
+
+　　有比停留在原地迷茫更糟糕的情况吗？
+
+　　最后，所有身外之物，生不带来、死不带走，尝试过所有的事情，不是本身就是一件很值得骄傲的事情吗？
+
+　　而且尝试新事物也是保持任何良好亲密关系的重要途径。
+
+　　动起来吧，也许某天，我们就会打开自己的新世界。
+
+　　南国微雪 Miyuki
+
+　　2025 年 11 月 17 日
\ No newline at end of file
```

```diff
@@ -0,0 +1,13 @@
+# 退路
+
+*刘墉*
+
+　　我曾在报纸上看过一则新闻，大意如下：
+
+　　有一家人遭了偷盗，为防止小偷再度破窗而入，屋主特别在楼下所有的窗子外面，加装一道铁栏。
+
+　　但是接着他住的二楼又进了小偷，并发现小偷是由一楼攀着窗上的铁栏进入二楼的。于是屋主又为二楼所有窗户装了铁栏，心想：我把每个窗子都装上铁栏，小偷就算再有本事，也不可能进来了吧？谁知不久以后，这家楼下突然失火，几个孩子和一个大人睡在二楼，因为铁栏的阻隔，无法逃生，竟攀在窗上，活活被烧死。
+
+　　当我们自以为断绝敌人的机会时，也可能正为他制造了另一个机会；而当我们把敌人所有的机会全部阻绝的时候，却可能也失去了自己的机会。
+
+　　不管我们怎么对付敌人，总得为自己留一步退路才行。
```
