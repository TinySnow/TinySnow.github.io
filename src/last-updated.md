+# 铁打的中产，流水的中产阶级
+
+*碧树西风 2025-12-22*
+
+　　又到年底了，我想借这个特殊的时间节点，补上多年以来，许多读者想要我写的一个话题。
+
+　　那就是上岸。
+
+　　从我少年时每天听到热议的下海，到我中年后，满耳朵听到的上岸，这真是很有趣的轮回。
+
+　　人们真有那么热衷于上岸后的工作内容本身么？
+
+　　我想不是的，人们真正热衷的，是中产这件事，是稳定这件事，是稳定的成为中产这件事。
+
+　　曾几何时，人们认为自己是中产阶级，就一直都会是，而且会步步高升，甚至冲一冲资产阶级。
+
+　　直到近些年，相当一部分人滑落后，恍然大悟：过手财富，和握手财富，是两码事。
+

+

+## 第一个话题：虚名与实利。
+
+　　人这辈子，一定要想清楚你要什么。你不要张口跟我讲东食西宿。东食西宿 = 自欺欺人，听得懂这句话么？

+　　我明明不优秀，但我就是希望七仙女爱上我，白素贞送我药铺送我银子送我飞，白马王子带我去中东做王妃。

+　　我明明又懒又无知，我就想加盟个奶茶店，人家品牌方替我操好心，我从此就躺赚，躺着赚品牌方的技术+员工的劳动力。
+
+　　结局就是你被骗嘛，没啥神奇的。
+
+　　品牌方赚你的加盟费，员工摸鱼赚你的工资，你掏钱，买一种你能够占便宜的感觉，最后，你来为这种感觉买单。
+
+　　感觉就叫虚名，买单就叫实利。
+
+　　所以这个世界上从来都存在两种东西，一种叫虚名，一种叫实利。
+
+　　比如在欧洲，大家都被称作 Mr. Miss。如果你想要换个头衔，称 Sir。需要有爵位，这个太难了，最简单的是读个博士，你就能加 Dr。衔了。
+
+　　你加了 Dr。终身都是 Dr。即便你早就忘了自己当年发表过什么论文，到死，你也可以要求别人称呼你时，加 Dr。这个就叫做虚名。
+
+　　你看国内很多昔日的教授，明明已经辞职了，明明已经下海做了网红艺人，但出场的时候，依然可以加教授头衔，就是 Prof。严格意义上，他们是前某某大学教授，但也没有人追究。
+
+　　因为不涉及实际利益。我称呼你 Prof。或者 Dr。顶多麻烦了点，谁也不会为此掉二两肉。
+
+　　假如你愿意把博士定义为一种阶层，把教授定义为一种阶层，那这就是一劳永逸，你做到一次，以后都可以使用。
+
+　　它天然就是稳定的。
+
+　　问题是，你安于此么？
+
+　　那个讲三国出名的什么教授，当年在某大的时候，一家人在 20 多平的教工宿舍里蜗居了十年，这是他自己说的。这就叫，有些阶层可以当饭吃，有些阶层不能当饭吃。
+
+　　所以当我们描述中产阶级这个话题时，首先要想清楚，我们在描述它的什么？
+
+　　很多国人热衷的中产阶级，实际上就是对占据利益的描述。
+
+　　一个家庭，有大几十万乃至上百万的年收入，有一二线全款的房产，有几百万的流动资金。

+　　这些东西在描述什么？统统在描述你所占据的社会资源。

+　　你的那个高薪，又不是大风刮来的，一个企业里面，又不可能每个岗位都拿高薪；一个行业里面，也不是每个企业都能发得出高薪。

+　　所以这时候就得出现一个岗位，叫监理。
+
+　　监理这个岗位，他不能是有利害干系的，不能说，装得好，发奖金有他一份，装得不好，扣他钱，不能这样。
+
+　　甚至他的考核体系都得是独立于该装修项目的。
+
+　　那么换言之，他拿的是什么？就是铁饭碗嘛。
+
+　　他的收入旱涝保收，他的收入不高不低，一定是这么设计的。
+
+　　这时候项目经理的内心就不安了，因为出现了他不可控的暗桩。
+
+　　一个项目经理不打报告都可以换掉泥工木工，但他换不掉监理。
+
+　　你换掉一个监理的人事任免权，也许要一路提交到公司董事会去。
+
+　　那个监理，他自己也很明白这一点。
+
+　　于是监理就会权衡，我配合项目经理，或许有好处，可是我要承担风险。
+
+　　可如果我的职业都这么稳定了，我干嘛还要承担这个风险呢？
+
+　　你去看多年前的，名义那本电视剧。
+
+　　里面就有这么一个很经典的场景，发生在李达康与孙连城之间。
+
+　　李达康最初就是准备强拆的，完全是因为现场陈岩石电话里的一句小金子，他瞬间 180 度大转弯。
+
+　　可是你想过没有，从法理上，你没法让山水集团再拿一笔钱。
+
+　　于是李达康就希望孙连城，一地二拍。
+
+　　他最初就这么设想的，但是孙连城死活不接这个锅。
+
+　　他当然没法接，哪怕你没学过法，你用脚趾头想想，也知道这事儿没法做呀。
+
+　　他又不是那个准备随时跑路的前任丁义珍。
+
+　　我们是站在观众的位置，开了上帝视角，你提前知道山水集团是坏的，所以你觉得李达康做法是对的。
+
+　　无论用什么方法，哪怕是非法的，只要针对坏人，你都叫好。
+
+　　但现实中呢？
+
+　　李达康自己都不知道山水集团是好是坏，他这种做事方式就能看出来，只要陈岩石嘴里冒出个什么，他随时都可以开搞。
+
+　　这就是一个经典的项目经理。
+
+　　他觉得既然我站在信息高地，我就要用足这个优势，去达成我想要达成的。
+
+　　如果他想要达成的是私利，他就可能变成祁同伟；即便没有私利，那也是他觉得，他为了你好。
+
+　　所以我说，这个人物角色，是一个经典的项目经理形象。
+
+　　他利用信息优势，即便不往兜里揣，也会站在他的视角下，去做他自己认为对业主有利的事情。
+
+　　孙连城这个角色，就更像一个监理。
+
+　　他觉得我们不能为了把事儿做成，就不合规矩，一女二嫁这总不能呀。
+
+　　于是他们俩人之间发生了剧烈的冲突。
+
+　　你去看剧中，李达康是非常厌恶孙连城的，他恨不能干掉他。
+
+　　但这不是私企，你干不掉他的。
+
+　　因为人家没有做错任何事，人家没有经济问题，也没有作风问题，也没有迟到早退，他做的每件事，流程上都找不到问题。
+
+　　这就是孙连城的底气。
+
+　　孙连城很清楚，李达康只能把自己调离，没可能动自己的职级。
+
+　　也就是说，他孙连城，可以被调到一个没有实权的岗位上，但他的职级还在，待遇不变。
+
+　　一旦李达康的野蛮生长出现了问题，他孙连城很可能就再次被启用了。
+
+　　这就是代理人困境的非对称博弈论下的相对解决方案。
+
+　　100 年前人类就研究明白了。
+
+　　你能做的就一件事，就是埋暗桩。
+
+　　只要动一个人的难度特别高，就会形成稳定预期，俗称铁饭碗。
+
+　　那么这些人里面，就总有不配合的。
+
+　　他们就形成了事实上的监理，事实上不配合的那个暗桩。
+
+　　你不要指望有完美的。没有完美的监理，这里的监理不是你期待的，帮你每天盯着工地，时刻跟你汇报项目经理有没有偷换材料。
+
+　　不是这个意思。
+
+　　事实上的监理的作用，更多的是给项目经理带来了掣肘，从而抵消了他的信息优势。
+
+　　因为他没法安心嘛，因为他发现工地上总有人收买不了。
+
+　　也有可能没较真，但万一要是较真了呢？或者私下打小报告了呢？
+
+　　那就会秋后算账，于是信息优势就被瓦解掉了一部分。
+
+　　你想通这一点，才能明白自己的饭碗因何而铁，以及成色几何。
+
+　　我 90 年代，还是个少年的时候，就是看报纸上，大人们的激烈辩论，看非对称信息优势下的博弈论。
+
+　　我看懂了。
+
+　　于是我对上岸这件事，有了一个正确的预期。
+
+　　我们院的小孩绝大部分都上岸了，我没有。
+
+　　因为我非常清楚这件事的预期收益是多少，都能算出来了。
+
+　　你想要的稳定，存在的前提是什么，你知道了。
+
+　　那你就会知道，这个稳定下的收益，它不会低。
+
+　　太低，你就抵挡不了项目经理的诱惑了。
+
+　　它也不会高。
+
+　　高了，你就飞走了，你就不肯兢兢业业当暗桩了。
+
+　　你去看雍正王朝里面佟国维的管家是怎么骂下人的？
+
+　　别把那鹰喂太饱了，饱了它就不捕猎了。
+
+　　也别饿着，饿着，它就啄你了。
+
+　　至于你说不清不楚的收入，那我告诉你，这就是一个数学期望的话题。
+
+　　早在 90 年代，社会上还挺乱的时候，我就想明白了，这就是数学期望。
+
+　　你的收益，要乘以风险，才是什么？才是净收益。
+
+　　我冒着杀头的风险得来的收益，它是收益么？它不是，因为它还没有乘以风险呢。
+
+　　乘完了最后剩下的，才是呀。
+
+　　你想想看，如果我都愿意冒着杀头的风险了，我干嘛不去金融市场里博弈呢？
+
+　　尽管那个年代我并不知道国际市场上杠杆最高可以 6000 倍。
+
+　　但我凭想象我也能想明白：
+
+　　如果都愿意冒险了，何必去做代理人呢？代理人代理人，不就是不想冒险么？
+
+　　…….
+
+　　多年后，2008 年的时候，我把 90 年代的这个想法，跟一个华尔街当时被辞退回国的交易员聊过。
+
+　　他大我挺多的，他就笑了。
+
+　　他说你理解的一点都没错，尽管 08 年我依然是个金融的外行，可我已经看到了代理人的本质。
+
+　　代理人，他就是不想冒险的。
+
+　　你赚一倍，他也赚你 1 个点的佣金，你亏一半，他也赚你 1 个点的佣金。
+
+　　旱涝保收，他才愿意当代理人嘛，否则他就自己冲进市场里博弈了呀。
+
+　　后来这些想法都是得到验证的。
+
+　　我 2011 年在甲方做了小一年，我亲眼见过很多同事，他就是这么自我定位的：
+
+　　我是代理人，我是孙连城，流程比结果重要。
+
+　　因为流程错了，我就完了，结果错了，我就跟金融市场上的代理人一样，还是吃一个点呀。
+
+　　无差别的呀。
+
+　　至此，我想我把上岸这件事最骨子里的底层逻辑，讲透了。
+
+　　我想我的比方很浅白，你们应该能听懂，无非就是能不能接受的问题。
+
+　　有些人，他不是听不懂，他是不愿意听。
+
+　　那我也没辙，没人能说服一个指望东食西宿求被骗的人。
+
+　　针对这部分人，我可以再讲一个小案例，让你们有直观的感受。
+
+　　我的前老大，当年我在甲方的时候，那位跺一脚，行业震三震的人。
+
+　　他现在在干嘛？他在踩缝纫机。
+
+　　以他的岁数，要出来，大约是来生了。
+
+　　我曾经私底下和他做过对比，比方方面面的能力，我都远不如他。
+
+　　因为我伺候过他，我明白彼此的能力差距。
+
+　　如果说他做过舰队的总指挥，我做过他的参谋，站在能力的角度，我清楚彼此的差距。
+
+　　但是，他犯错误的那点钱，都没我投资挣的多。
+
+　　这里面不是一个能力问题，你的能力再大，你是做代理人的。
+
+　　你那钱，拿了是有问题的。
+
+　　我能力再弱，可我是做高频交易的。
+
+　　我承担了风险，我用的是自己的钱，那我赚的没毛病呀。
+
+　　看到问题了么？
+
+　　当你个人能力特别优秀的时候，你选择做代理人，对你个人而言，在金钱上，肯定是不划算的。
+
+　　所以我 2012 年去创业的时候，就是这么想问题的。
+
+　　我待在甲方，单纯看金钱这个视角，它就是注定没法划算的。
+
+　　而我这个人又这么庸俗，又不看重地位的。
+
+　　既然我明明一个人就能做项目，我干嘛要替你挣钱呢？我干嘛要给你当那个项目经理呢？
+
+　　就像我明明会投资，我干嘛要替你投资，你动动脑筋想想嘛。
+
+　　资金不够可以借的，可以加杠杆的，只要付出一点利息就可以了，没谁傻到愿意支付利润，如果他真有的话。
+
+　　他愿意支付利润就一个原因，那就是小沈阳说的，这个真没有嘛。
+
+　　他根本不确定他能赚钱，所以他才肯拿你的钱，去赌嘛。
+
+　　是不是这个理儿？
+
+　　当然，我讲这些，不是给前上司开脱，他活该，他就该踩一辈子缝纫机。
+
+　　这个无可辩驳。
+
+　　我只是说，即便像他这样的，会赚钱，能赚钱，也真的为甲方赚过大钱的人。
+
+　　最后自己拿走了其中很少的一点点，也得踩一辈子缝纫机。
+
+　　这就叫代理人游戏。
+
+　　你自己想清楚，没有东食西宿，稳定和高收益，你没法兼得。
+
+　　内地的女明星，刚刚去世的演过四大名著里各种美女的那位，她的遗产，只有 1000 个。
+
+　　为什么？
+
+　　因为她当年拒绝了琼瑶的橄榄枝，她拒绝下海，她要留在原单位。
+
+　　于是她得到了稳定的同时，也就失去了与她那个体量相衬的明星该有的收入级别。
+
+## 第三个话题：不是每个人都适合上岸。
+
+　　其实通过上一个话题，我们已经能够看到，有一种人是明显不适合上岸的。
+
+　　你就比如我。
+
+　　我 2011 年在甲方是被半年之内连升三级，各种重用加挽留的。
+
+　　也留不住。
+
+　　因为我就不是一个说两句好听话，就能留得住的人。
+
+　　我是个纯粹的生意人，当我知道自己能赚钱的时候，我就不乐意给别人赚钱。
+
+　　这种就不适合上岸。
+
+　　你真的会赚钱，你眼里又只有钱，那你真的不适合上岸。
+
+　　你很可能会害了自己，你很可能某一天觉得你作为代理人都替东家赚了天文数字了，自己拿一丁点怎么了。
+
+　　法律上可不这么看，法律上那东西是写死的，你觉得一丁点是相对于你自己替东家赚过的钱。
+
+　　法律上，可不认为那是一点，法律上认为那个数字很大，足以让你终身踩缝纫机了。
+
+　　所以我们反推就知道，什么人最适合上岸？
+
+　　很简单，不会赚钱的嘛。
+
+　　你像我妈妈这种，我太太这种，就都很合适呀。
+
+　　她们的思维模式真就是孙连城式的，她不会给你惊喜，也不会给你惊吓。
+
+　　你用流程来考核，那就是认认真真，兢兢业业。
+
+　　但是你把她放在私企的考核体系里，马上就被干掉了。
+
+　　因为私企老板要的是结果，不是流程。
+
+　　所以我总结一下，我从小生活的环境，我的亲戚有一个算一个，都是上岸的。
+
+　　我小时候的邻居，有一个算一个，都是上岸的。
+
+　　包括我后来几十年对上岸的观察，我认为一个人具备以下三个特征，是更容易适应岸上生活的。
+
+### 第一，你要耐得住。
+
+　　有一部剧，把耐得住三个字讲得非常透，就是士兵突击。
+
+　　许三多被安排去五班看守草原，和老兵油子李梦对话。
+
+　　三多说，来之前，连指导员告诉他，看守草原很有意义，这是一个光荣且艰巨的任务。
+
+　　李梦就讲，等你待久了，你才会理解这五个字：
+
+　　光荣在于平淡，艰巨在于漫长。
+
+　　天底下不是所有人都耐得住的，草原五班包括班长在内，另外四个都受不了，只有许三多，是真适合。
+
+　　他后来一个人看守连部都能看半年之久，你让我干这活儿，我绝对疯掉了。
+
+　　我不止一次收到过来自上岸的读者的留言。
+
+　　你比如我摘一条。
+
+　　有个满级读者，曾经留言说：他上岸了之后，就老琢磨一件事，自己当初为啥要上岸？
+
+　　他以前是私企的，每天工作很辛苦，尤其是加班到很晚，再坐地铁晚班车的感觉，真不好。
+
+　　所以上班时间就花精力复习考试，终于上岸了。本以为迎接自己的，是如意的生活。
+
+　　起初的一个月，也很开心，觉得节奏慢下来了，压力没那么大了，食堂的饭菜也格外香。
+
+　　但渐渐的，一年两年三年，越来越有一种乏透了的感觉。
+
+　　以前是身累，现在是心累，身体累，还可以周末补个觉，心累是晚上睡都睡不着。
+
+　　身体累是我知道问题在哪儿，这个工作量太大了，节奏太快了，我做不完，我跟不上，我好歹知道咋回事。
+
+　　心累呢，是我不明白问题在哪儿，我浑身的力气，使不上劲。
+
+　　我就是觉得累，可我又不知道这种无影无踪的累，打哪儿来的，如何消除…….
+
+　　你可以看到，这就是一个经典的耐不住的案例。
+
+　　说到底，你还是拎不清监理与项目经理的角色。
+
+　　稳定，是监理的身份带给你的。
+
+　　成就，是项目经理的身份带给你的。
+
+　　你老想着东食西宿，你就焦虑，你就内耗，你就痛苦。
+
+　　我讲一个故事。
+
+　　上一期，我讲了一个 35 岁遇到公司被收购的业务主管老真田的故事。
+
+　　这个故事有原型的，原型人物就是 400 多年前，日本战国的名将，真田昌幸。
+
+　　昌幸之前是跟武田家混的，后来武田家被灭，他就独立成小大名，在乱世中苟活。
+
+　　你玩过太阁立志传游戏就知道，这哥们很牛的，打仗很牛，智谋更牛。
+
+　　那么后来，丰臣秀吉统一了日本，国内再无战争，真田昌幸呢，也就成了丰臣的家臣之一。
+
+　　太平的日子，相对于战争，可不就是上岸了么？
+
+　　真田昌幸这段时光，应该过得很幸福吧？
+
+　　恰恰相反。
+
+　　为什么？
+
+　　因为日常的工作从战争，变成了内斗，丰臣家的大秘石田三成，和高管德川家康之间的内斗；老板丰臣秀吉，和外甥丰臣秀次之间的内斗。
+
+　　你可以看到很有意思的，打仗的时候大家都会想，不打了，咱们好好过日子吧。
+
+　　可当真不打了，人们并不会好好过日子的。
+
+　　这话是任我行跟令狐冲说的，天底下并没有什么世外桃源牛背山，你去哪座山，都是人和人之间在斗来斗去。
+
+　　没有人能退出江湖，因为有人的地方，就有江湖，什么是江湖？人心的浮动才是江湖。
+
+　　打仗的时候，就像小公司。小公司难不难？很难。
+
+　　因为吃喝拉撒都是问题，活不活得下去，提心吊胆。
+
+　　内斗的时候，就像大公司，大公司难不难？也难。
+
+　　就是红楼梦里面王熙凤对刘姥姥说的那番话，大有大的难处。
+
+　　你看贾府里面，几百口子下人，成天都在王熙凤这里打马虎眼，一个个各怀鬼胎。
+
+　　王熙凤忙得晨昏颠倒，饭都顾不上吃一口。
+
+　　用今天的话讲，你们家买了很多扫地机器人，按理说是它伺候你，实际上呢，是你在操心它。
+
+　　真田昌幸进入丰臣时期之后的处境就是这样。
+
+　　今天领导交代一个茶话会，你有没有组织好？
+
+　　会议上摆着水果，这个橘子的产地是哪里的？
+
+　　哦，产地是你老家的，这怎么回事？你给说明一下，为什么采购你老家的橘子？这里面会不会有猫腻儿？
+
+　　这个花生从哪个供应商那里进的？这个供应商有没有查三代？他三代之内，和你真田家，是不是有交集？
+

+# 椰子树与平等
+
+*王小波*
+
+　　二十多年前，我在云南插队。当地气候炎热，出产各种热带水果，就是没有椰子。整个云南都不长椰子，根据野史记载，这其中有个缘故。据说，在三国以前，云南到处都是椰子，树下住着幸福的少数民族。众所周知，椰子有很多用处，椰茸可以当饭吃，椰子油也可食用。椰子树叶里的纤维可以织粗糙的衣裙，椰子树干是木材。这种树木可以满足人的大部分需要，当地人也就不事农耕，过着悠闲的生活。忽一日诸葛亮南征来到此地，他要教化当地人，让他们遵从我们的生活方式：干我们的活，穿我们的衣服，服从我们的制度。这件事起初不大成功，当地人没看出我们的生活方式有什么优越之处。首先，秋收春种，活得很累，起码比摘椰子要累；其次，汉族人的衣着在当地也不适用。就以诸葛先生为例，那身道袍料子虽好，穿在身上除了捂汗和捂痱子，捂不出别的来；至于那顶道冠，既不遮阳，也不挡雨，只能招马蜂进去做窝。当地天热，搞两片椰树叶把羞处遮遮就可以了。至于汉朝的政治制度，对当地的少数民族来说，未免太过烦琐。诸葛先生磨破了嘴皮子，言必称孔孟，但也没人听。他不觉得自己的道理不对，却把账算在了椰子树身上：下了一道命令，一夜之间就把云南的椰树砍了个精光；免得这些蛮夷之人听不进圣贤的道理。没了这些树，他说话就有人听了——对此，我的解释是，诸葛亮他老人家南征，可不是一个人去的，还带了好多的兵，砍树用的刀斧也可以用来砍人，砍树这件事说明他手下的人手够用，刀斧也够用。当地人明白了这个意思，就怕了诸葛先生。我这种看法你尽可以不同意——我知道你会说，诸葛亮乃古之贤人，不会这样赤裸裸地用武力威胁别人；所以，我也不想坚持这种观点。
