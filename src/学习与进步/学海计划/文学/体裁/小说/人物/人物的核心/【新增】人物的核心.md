## 正文

　　本文讨论：文学-体裁-小说-人物-人物的核心。

　　如果只能用两个问题来认识一个人物，这两个问题是：<u>**他想要什么？他恐惧什么？**</u>欲望和恐惧是人物心理的两个极点，就像磁铁的正负极——所有行为都在这个磁场里运动。想要的东西指明了人物运动的方向，恐惧的东西划定了人物不会跨越的底线——而中间那片灰色地带，就是故事情节发生的全部空间。

　　但这里有一个更精细的区分，我自己管它叫<u>**"want 和 need 的剪刀差"**</u>：want 是人物意识到的、主动追逐的目标——"我要得到黛西"、"我要成为上流社会的一员"；need 是人物自己都没有意识到、但真正匮乏的东西——"我需要被承认为有价值的人"、"我需要在不完美的人生里找到和解的可能"。盖茨比的 want 是黛西——他以为只要夺回黛西，人生就完整了。盖茨比的 need 是消解那个贫穷的、被社会拒绝的詹姆斯·盖茨（他的原名），成为另一个自己——不是盖茨比拥有了黛西就完整了，而是盖茨比需要用"拥有黛西"来证明"我成功了、我值得被爱"。want 和 need 之间的错位越大，人物的内在张力就越强。而最伟大的悲剧往往发生在人物得到了 want 却发现那根本不是 need 的时候。

　　《红与黑》里的于连·索雷尔，表面 want 的是社会地位和贵族爱情，深层 need 的是被世界承认自己的价值。他的两种 want——通过教会（黑色道袍）和通过女人（瑞那夫人和拉莫尔小姐）攀升——最终一个都没有真正满足他的 need。不是因为失败，而是因为即使成功了（当上了轻骑兵中尉、被拉莫尔侯爵认可），他也发现那并不是他要的。那个在断头台上回顾一生的于连，终于看清了：他一直用"征服世界"来补偿"出身低微"的内伤——但征服永远填不平那道伤。这就是经典人物的核心结构：**一个他自己都弄不懂自己的灵魂。**

　　对于写作者，一个很实用的建议：在动笔之前，用一张便签纸写下你的主人公的 (1) 自觉欲望（want），(2) 不自觉的真正需求（need），(3) 他最恐惧的事。如果这三个答案之间有清晰的张力——比如 want 和 need 方向相反，或者他最恐惧的事恰恰是满足 need 必须面对的事——你的人物就立住了。如果没有——故事可能会卡在半路，因为人物没有足够的内部推力来驱动情节。
