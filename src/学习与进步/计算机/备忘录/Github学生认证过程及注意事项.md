# Github 学生认证过程及注意事项

1. 登录四川农业大学[学生自助打印云平台](https://dzzm.sicau.edu.cn:30813/transcript_server/login)，选择校友，登录，选择成绩等级证明（因为这个证明文件生成的时间是当天，所以可以伪造自己在学校里）

2. 按照其他博客的方法，填好 Github 上相关必填的信息，推荐[这篇博客](https://blog.csdn.net/qq_46106285/article/details/149153111)，为防止链接失效，简单写一下需要完成什么事情：

    - 修改 Github 档案的 Public Profile，里面的 Name 为中文拼音，顺序因为英语名字顺序，如“王二麻子”则填写为“Ermazi Wang”，认证通过之后可以改回来
    - 修改 Github 档案的 Billing and licensing -> Payment information，里面 First name, Last name 与 Public name 保持一致，Address (Street, P.O. box) 直接填 City 的值，City 填学校所在城市的英文拼音（如“Ya'an”），Country/Region 选中国
    - 梯子的 ip 最好改成香港，反正离学校越近越好，最好不用梯子，当然有学校 VPN 更好，推荐手机流量认证，如果硬是进不去，那就只能改 Hosts 加 Github 的 ip 了

3. 手机认证，浏览器进入 Github -> Billing and licensing -> Education benefits -> Start an application

4. 如果 Github 添加并认证了校园邮箱，直接选择学校即可，如果没有自己手动选一下

5. 选 Student，然后 Share Location 点一下，分享自己的位置

6. 证明材料选第二个 2. Dated official/unofficial transcript 成绩单，电脑打开这个文件，切一下后置镜头，手机拍电脑屏幕，因为 Github 拍照之后会压缩，像素会非常烂，拍完之后自己看一下，尽量拍清楚一点

7. 第一次做完这一步就没了，如果问你为什么没有在学校里（Why are you not on campus?），看一下网上怎么弄的吧，大多数都是加了远程学习（Distance Learning）就过了，因为一定需要一张图片作为佐证，可以 F12 控制台 p 一张，或者 A4 纸手写一个证明材料，网上有相关的教程
