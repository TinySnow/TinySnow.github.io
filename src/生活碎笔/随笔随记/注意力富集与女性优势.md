# 注意力富集与女性优势

　　这是一篇实用类随记，不过我估计写得不会很长，因为概念太简单了。我想做的只是纠正看法。

　　社会对于男女两性的期望不太相同，一般来说，对于男性是工具性，对于女性是表达性，转换一下说法，就是要求男性“很好用”，而要求女性“很好看”或者“很会表达”。

　　我认识的有些女性朋友，比较抗拒这种社会期望，认为是“服美役”。

　　当然能够理解。就跟上来要求男生有房有车有存款一样，任谁都抗拒。

　　不过我今天想说的是另一面。希望能有所启发吧。

　　生物上有一个效应，叫富集作用。讲的是自然环境中的化学物质浓度沿生物链在各级生物体内逐渐递增的现象，有害的物质一旦进入生物体内，就不容易排出，只进不出，一步一步的叠加，最终在生物链的最顶部的生物体内的含量可能达到比原始环境高百万级别的浓度。

　　在注意力上面其实也存在有这种情况。

　　想象一个情况，我们稍微物化一下任何人，把每个人的长相从 1-10 分排列起来。把分值相同的一对男女排在一起，你会最先看到谁？

　　一般都是女生。

　　不过跟富集作用不同，越往上这种情况会越往男性身上靠，毕竟帅哥太少了，而美女相对而言没有那么少。越往上富集作用越不明显，稀缺性带来的效应越明显。这个富集作用解决不了。如果单拎一个性别的话，那肯定毫无疑问是看到更好看的人。

　　但是，大多数情况下，我们都是普通人，不是眉清目秀大帅哥或者倾国倾城大美女。

　　所以在日常/普通条件下，女性获得的注意力，肯定是要远大于男性的。

　　一般女生可能会觉得这是一种压力，但是我们可以反过来想，这也可以是一种优势，或者动力。

　　尤其是在虚拟主播赛道上，你可以看到没几个男的去做 Virtual Youtuber（虚拟主播）的，去做这个东西的男的，大多数也都没什么人气。但是受人追捧的女性 VTB 一大把。

　　而虚拟主播一般只有一个皮套，画师画的，看不到脸的。

　　这是人们注意力分配上比较极端的一个例子，也是一个现状。

　　我不知道看到这儿的女性读者有多少，不过如果你愿意冲进那些性别比例差距比较大的群体中的话，一般来说能获得更多的资源和关注。当然，这个赛道最好不要有更多的同性关注到，不然你拉不下脸，那可有的是人拉得下脸。

　　这是天然的优势。

　　比如一个女销售天然就比一个男销售更有亲和力。

　　不过有优势也就有代价，看看自己能不能承受。

　　双刃剑一面朝向别人，一面朝向自己。获得更高注意力的代价，就是自己有更高的可能“只看到外表”。然后随之而来的，职场歧视、不合群、各种乱七八糟的东西。这是相生相伴的。

　　可能有人看到阴暗面会觉得，啊，好难啊，还不如不去呢。

　　但是不去的话，总会有人跟你竞争搏斗的。就待在女性赛道的话，比如美妆啊、旅游啊（发现没有，旅游博主你们发现有几个男的，几个女的呢？）、宠物啊，这点优势也会被抹平的。毕竟如果有人在教妆造的时候，说自己是女生怎么怎么样，不论是男是女都会觉得比较蠢吧。这就是优势抹平了。

　　也有人觉得会觉得这是在物化怎么怎么样的。我不置可否。

　　另一边的人都已经被物化习惯了。工具人嘛。搞那么好看干嘛，在外面搞钱就行了。

　　不过说实话，这种把男女划成两个阵营、两个世界的结果无论对男对女都不好，毕竟多一个朋友总比多一个对手好，我们总是希望对方跟自己有更多的共同语言。

　　但是这个都扯远了，这是另一个社会学话题了。我们今天聊的只是认识。

　　在社会现状如此之下，我们怎样去发掘我们的优势。而至于这个社会现状好不好、要怎么去改，那是另外的东西。

　　如果真的能够处理好伴随而来的代价，运用这个天然的优势，能够带来丰厚的回报。

　　注意力富集而来的东西，可能不仅仅是职级、薪酬，还有更多的人际关系、消息面和新的自己可能未曾接触过的异性世界。

　　这个我可以保真，毕竟我自己主动接触化妆、心理等等东西之后，很多人都表示诧异，觉得我和其他男的不太一样。

　　换位思考之后结果一样的。

　　如果有人愿意跟我一起打游戏，拼模型，玩摄影，聊到车有聊不完的话题，我也会觉得这个人跟其他女生不一样，很懂我们。

　　在这点上，无论男女其实都一样的。只是看自己愿不愿意接触对方的世界而已。

　　如果女性愿意这么做的话可能回报更丰厚，毕竟男的是真好用吧，而不仅仅是好看（笑。

　　说到底，优势用不用是自己的决定，但优势存在不存在，这个是客观的。

　　而且这个也对自己是一种考验，毕竟躲在优势处偷懒的人大家都不喜欢。明明自己搬得动的水非要你搬。

　　毕竟我们的前提是：分值相同的情况下。

　　最后说一句，无论怎么做，都可以想想自己的优势、劣势，自己想要什么，能舍弃什么。

　　毕竟“人际和谐”、“回报公平”、“变得强大”是一个不可能三角，不可能三个都要。
