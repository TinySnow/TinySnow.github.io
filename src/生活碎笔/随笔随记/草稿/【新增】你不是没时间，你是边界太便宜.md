# 你不是没时间，你是边界太便宜

　　我最烦听人说一句：“我太忙了，真没时间。”

　　不是我不信——是我太信了所以觉得没意思。你当然忙，谁会一天到晚闲着。

　　但问题从来不是你忙不忙，是你为什么永远在忙别人的事情，然后自己的事情永远在等。

　　你仔细想：今天你做的所有事，有多少是自己真正决定要做的？有多少是别人塞给你的？

　　那个比例决定了你到底是在经营自己还是被经营。

## 一、你的时间有一个价格，但你没定过

　　大多数人的时间属于**无标价的公共资源**，谁先伸手，谁就占了。

　　同事说“帮我看一眼这个”，你说“好”。群里@你说“谁有空做个会议纪要”，你说“我来”。

　　你看，你做这些决策的时候没花一秒钟。但你自己的论文/代码/方案，拖了三个月。

　　你知道为什么吗？因为拒绝别人是一个有心理成本的动作，而拖延自己的事是一个无感知成本的动作。

　　所以你的默认策略永远是：先把别人打发走，自己往后排。

　　**你的时间不是不够用，是你的时间单位没定价。没定价的东西，人就会无限侵占。**

## 二、一个让你不太舒服的真相

　　你的时间之所以这么便宜，是因为你自己都没觉得它值钱。

　　说真的，你不觉得自己在“免费帮人”的可不只是别人，也包括你自己。

　　你自己都在用最廉价的方式花掉自己的时间——刷不完的信息流、回复不完的“在吗”、参加不完的“其实没必要的会”。

　　如果你自己都在用超市清仓的方式处理你的时间，别人凭什么给你出专柜价？

　　你想想，你觉得你很忙，但如果我站你身后看四十个小时，有几件是真的不干就会死的事？

## 三、边界不是不帮忙，是帮你算账

　　很多人说“我没法拒绝，因为不想显得不合群。”

　　我理解这个压力。但你换个算法。

　　你花了四十分钟帮同事改了一个他完全能自己搞定的东西。

　　这四十分钟如果按你的时薪算，值多少钱？按你未来的时薪算呢？

　　你是花了这笔钱请同事吃饭了吗？没有，这顿饭他都不知道你请了。

　　这是单向、隐形、零回报的价值转移。

　　我不是让你变成一个斤斤计较的铁公鸡。我是说：**你的善意如果没有边界，就会变成别人默认的税。**

　　你每年给世界交的时间税，可能比你交的个人所得税还高。

## 四、边界要提前设，不是事后怨

　　最无效的边界管理是：你帮完了，回到家跟家人抱怨“烦死了天天找我干活”。

　　你怨了，对方不知道。下次还是找你。

　　工具里有一个概念叫 **API 定义**：系统能对外提供什么服务、用什么格式、什么时间可调用，全写清楚，不写在文档里的就等于没有。

　　你的时间也需要一个 API 定义。不是说“我不帮忙”，是说“什么范畴的我可以帮、什么时间段我必须做自己的事、什么类型的请求我不处理”。

　　这个定义不需要写成文档发给全公司，但它需要在你脑子里明确到你可以脱口而出。

## 五、今天就能改的三件事

　　第一，你不想去的一个酒局，直接说“今晚有安排了”——不需要解释安排是什么。**不需要解释**。

　　第二，别人叫你干一个你不想干又不是你职责的事，回一句“我现在手头有个东西赶着交付，你找 XX 或者自己先试试，不行我再帮你看”。

　　第三，每天晚上复盘一下：今天有哪段时间是我主动分配的？哪段时间是被动卷走的？

　　这三个动作不进肌肉记忆，你永远是被牵着走的那一个。

　　你的时间是黄金还是散沙，不是因为它本身值多少，是因为你有没有给它围上一道墙。

　　你先围上，别人才会排队。不围，就是随便捡。

　　这道理简单到让人觉得不好意思——但大多数人一辈子都没对自己说过一句"我的时间现在值这个价"，对吧。
